# 🌐 Personal Portfolio Website

A modern and interactive portfolio website showcasing my projects, skills, and interests in web development and cybersecurity.

---

## ✨ Features

- 🎨 Interactive animated background (Three.js tubes cursor effect)
- 🧑‍💻 Clean and responsive portfolio layout
- 📂 Dedicated projects section with live links
- 🛠️ Structured skills and technologies overview
- 📬 Contact section with social links
- ⚡ Smooth UI with glassmorphism styling

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Three.js (for interactive background)
